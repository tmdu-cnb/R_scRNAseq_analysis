# Purkinje細胞限定マトリックス

## 内容

- **DC_PURKINJE_counts.csv** - DC条件のPurkinje細胞限定カウントマトリックス
- **DC_PURKINJE_counts_with_ids.csv** - ID付き版
- **DT_PURKINJE_counts.csv** - DT条件のPurkinje細胞限定カウントマトリックス
- **DT_PURKINJE_counts_with_ids.csv** - ID付き版
- **KC_PURKINJE_counts.csv** - KC条件のPurkinje細胞限定カウントマトリックス
- **KT_PURKINJE_counts.csv** - KT条件のPurkinje細胞限定カウントマトリックス

## 説明

生データからPurkinje細胞を抽出して作成したカウントマトリックスです。DEG解析に使用されます。
