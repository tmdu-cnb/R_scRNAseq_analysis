# 修論用：細胞タイプ比率解析の方法論説明

## 【要約】細胞構成・相対比率・比較・統計処理方法

### 細胞構成の相対比率比較における統計処理方法

本研究では、**細胞構成（cellular composition）**の変化を評価するため、各細胞タイプの**相対比率（relative proportion）**を群間で**比較（comparison）**した。細胞タイプ比率は構成データの性質を持つため、従来の統計手法では適切に比較できない。そこで、**ディリクレ多項式モデリング（Dirichlet Multinomial Modeling, DMM）**を用いた**統計処理方法（statistical analysis method）**を採用した。

#### 統計処理の流れ

1. **細胞構成の定量化**：各サンプルにおける10種類の小脳細胞タイプの細胞数をカウント
2. **相対比率の推定**：多項分布とディリクレ分布による階層的ベイズモデルで各群の細胞タイプ比率を推定
3. **群間比較**：Log2(FC)の事後分布を計算し、DTA群とCTL群の相対比率を比較
4. **統計的有意性の判定**：89%ベイズ信用区間により有意差を判定

#### 統計処理方法の特徴

- **構成データの制約を考慮**：負の共分散構造を統計的に正しく扱う
- **サンプル間のばらつきを考慮**：生物学的な変動をモデルに組み込む
- **ベイズ推定**：ハミルトニアン・モンテカルロ（HMC）法による事後分布の推定
- **不確実性の定量化**：89%ベイズ信用区間による信頼性の評価

## 【修論本文用】細胞タイプ比率解析セクション

### 目的

次に、DTAモデルにおける小脳細胞組成の変化を明らかにするため、DTA群と対照群（CTL）間で各細胞タイプの比率を比較した。

### 原理：構成データの統計的性質とモデリングの必要性

#### ① 構成データの制約

細胞タイプ比率は**構成データ（compositional data）**の性質を持つ。構成データとは、各要素の合計が一定（100%または1）という制約を持つデータである。この制約により、ある細胞タイプの比率が増加すると、他の細胞タイプの比率は必ず減少するという**負の共分散構造**が生じる。

#### ② 単純な統計手法の問題点

従来の統計手法（t検定、カイ二乗検定など）を構成データに直接適用すると、以下の問題が生じる：

1. **見かけ上の負の相関**：実際には変化していない細胞タイプまで、見かけ上「減少した」ように見える
2. **多重比較の問題**：細胞タイプ間の依存関係を考慮できない
3. **サンプル間のばらつきの無視**：各サンプル内の細胞数や比率の変動を適切にモデル化できない

#### ③ 多項分布とディリクレ分布による階層的モデリング

細胞タイプ比率の統計的モデリングには、以下の2層構造を仮定する：

**第1層：サンプル内の変動（多項分布）**

各サンプル$s$において、観測された細胞タイプ別細胞数$\mathbf{n}_s = (n_{s1}, n_{s2}, \ldots, n_{sK})$は、そのサンプルの真の細胞タイプ比率$\boldsymbol{\theta}_s = (\theta_{s1}, \theta_{s2}, \ldots, \theta_{sK})$に基づく多項分布に従う：

$$\mathbf{n}_s \sim \text{Multinomial}(N_s, \boldsymbol{\theta}_s)$$

ここで、$N_s = \sum_{k=1}^{K} n_{sk}$はサンプル$s$の総細胞数、$K$は細胞タイプ数である。

**第2層：群内の変動（ディリクレ分布）**

各群（DTA群またはCTL群）内において、サンプル間の細胞タイプ比率の変動はディリクレ分布に従う：

$$\boldsymbol{\theta}_s \sim \text{Dirichlet}(\boldsymbol{\alpha})$$

ここで、$\boldsymbol{\alpha} = (\alpha_1, \alpha_2, \ldots, \alpha_K)$はディリクレ分布のパラメータ（濃度パラメータ）である。

この階層的モデルにより、以下の情報を同時に推定できる：

- **比率の変動**：各群における細胞タイプ比率の平均とばらつき
- **サンプル間ばらつき**：生物学的なばらつきを考慮した不確実性の評価
- **細胞タイプ間の相対関係**：構成データの制約を統計的に正しく扱う

### 方法：ディリクレ多項式モデリング（DMM）

#### ① データの準備

各サンプルにおいて、10種類の小脳細胞タイプ（顆粒細胞、プルキンエ細胞、ベルクマングリア、介在ニューロン、アストロサイト、オリゴデンドロサイト、OPC、ミクログリア、内皮細胞、線維芽細胞）の細胞数をカウントし、サンプル×細胞タイプのカウントマトリックスを作成した。

#### ② 事前分布の設定

ディリクレ分布の事前分布パラメータとして、$\alpha_k = 10^{-7}$（$k = 1, 2, \ldots, K$）を設定した。これは、各細胞タイプに対して均一かつ弱い情報事前分布（weakly informative prior）を仮定することを意味する。

#### ③ ハミルトニアン・モンテカルロ（HMC）法による事後分布の推定

パラメータ推定には、**ハミルトニアン・モンテカルロ（Hamiltonian Monte Carlo, HMC）法**を適用した。HMCは、マルコフ連鎖モンテカルロ（MCMC）法の一種であり、勾配情報を利用して効率的に事後分布からサンプルを生成する。

Stan（rstanパッケージ）を使用し、以下の設定でHMCサンプリングを実行した：

- **チェーン数**：4
- **反復回数**：4,000回（ウォームアップ：2,000回）
- **適応パラメータ**：`adapt_delta = 0.95`

各群（DTA群、CTL群）について独立に事後分布を推定し、各細胞タイプの比率$\theta_k$の事後分布を取得した。

#### ④ 群間比較：Log2(FC)の事後分布の計算

群間差の評価には、**Log2(FC)**（Log2 Fold Change）の事後分布を計算した。Log2(FC)は以下のように定義される：

$$\text{Log2(FC)}_k = \log_2\left(\frac{\theta_{\text{DTA},k}}{\theta_{\text{CTL},k}}\right)$$

ここで、$\theta_{\text{DTA},k}$と$\theta_{\text{CTL},k}$は、それぞれDTA群とCTL群における細胞タイプ$k$の比率の事後分布から得られたサンプルである。

各HMCサンプルについて、DTA群とCTL群の対応するサンプルからLog2(FC)を計算し、Log2(FC)の事後分布を構築した。

#### ⑤ 有意性の判定：89%ベイズ信用区間

推定値の不確実性は、**89%ベイズ信用区間（89% credible interval）**により評価した。89%信用区間は、事後分布の5.5パーセンタイルと94.5パーセンタイルにより定義される。

有意性の判定基準は以下の通りである：

- **有意な増加**：Log2(FC)の89%信用区間の下限が0より大きい
- **有意な減少**：Log2(FC)の89%信用区間の上限が0より小さい
- **有意差なし**：Log2(FC)の89%信用区間が0を含む

### 結果の記述例

その結果、**顆粒細胞**はDTA群において有意に減少していた（Log2(FC) = -X.XX [89% CI: -X.XX, -X.XX]）。一方で、**プルキンエ細胞**については有意ではないものの、減少傾向が認められた（Log2(FC) = -X.XX [89% CI: -X.XX, X.XX]）。これは、個々のサンプルのばらつきや希少細胞タイプの変化を検出する力の低下によるものと考えられる。

さらに、**アストロサイト**、**ベルクマングリア**、**ミクログリア**、**オリゴデンドロサイト**は、対照群と比較してDTA群で有意な増加を示した（図○、89%信用区間）。

## 【補足説明】技術的詳細

### Stanモデルの構造

本研究で使用したStanモデルは以下の通りである：

```stan
data {
  int<lower=1> N;  // サンプル数
  int<lower=1> K;  // 細胞タイプ数
  int<lower=0> counts[N, K];  // カウントマトリックス
  real<lower=0> alpha_prior;  // ディリクレ事前分布のパラメータ（10^-7）
}

parameters {
  simplex[K] theta;  // 細胞タイプの割合（simplex制約: 合計=1）
}

model {
  // ディリクレ事前分布（均一な事前分布）
  theta ~ dirichlet(rep_vector(alpha_prior, K));
  
  // 多項式分布で各サンプルをモデル化
  for (n in 1:N) {
    counts[n] ~ multinomial(theta);
  }
}
```

### 計算の流れ

1. **データ読み込み**：処理済みSeuratオブジェクト（`seu_dta_processed.rds`, `seu_ctl_processed.rds`）から各サンプルの細胞タイプカウントを抽出
2. **カウントマトリックスの作成**：サンプル×細胞タイプのマトリックスを作成
3. **Stanモデルのコンパイル**：Stanモデルコードをコンパイル
4. **HMCサンプリング**：各群について独立にHMCサンプリングを実行
5. **事後分布の抽出**：各細胞タイプの比率$\theta_k$の事後分布サンプルを抽出
6. **Log2(FC)の計算**：対応するサンプルからLog2(FC)を計算
7. **信用区間の計算**：89%ベイズ信用区間を計算し、有意性を判定

### 使用ソフトウェア

- **R** (version X.X.X)
- **rstan** (version X.X.X): StanのRインターフェース
- **bayestestR** (version X.X.X): ベイズ統計解析の補助関数
- **Stan** (version X.X.X): 確率的プログラミング言語

## 【キーワード別詳細説明】

### 1. 細胞構成（Cellular Composition）

**定義**：組織や臓器における細胞タイプの種類とその存在量の組み合わせ。

**本研究での扱い**：
- 10種類の小脳細胞タイプ（顆粒細胞、プルキンエ細胞、ベルクマングリア、介在ニューロン、アストロサイト、オリゴデンドロサイト、OPC、ミクログリア、内皮細胞、線維芽細胞）を対象
- 各サンプルにおける各細胞タイプの細胞数をscRNA-seqデータからカウント

**重要性**：疾患や処置により、特定の細胞タイプが増加または減少することで、組織全体の機能が変化する。

---

### 2. 相対比率（Relative Proportion）

**定義**：各細胞タイプが全体に占める割合。総和が1（100%）という制約を持つ。

**計算方法**：
$$\text{相対比率}_k = \frac{\text{細胞タイプ}k\text{の細胞数}}{\text{全細胞数}}$$

**特徴**：
- **構成データの性質**：ある細胞タイプの比率が増加すると、他の細胞タイプの比率は必ず減少する
- **負の共分散**：細胞タイプ間に負の相関が生じる
- **絶対数との違い**：相対比率は細胞数の絶対値に依存しないため、サンプル間の比較が可能

**本研究での扱い**：
- 各サンプルにおける相対比率を多項分布でモデル化
- 各群における相対比率の分布をディリクレ分布でモデル化

---

### 3. 比較（Comparison）

**目的**：DTA群とCTL群における細胞構成の違いを定量的に評価する。

**比較指標**：**Log2(FC)**（Log2 Fold Change）

$$\text{Log2(FC)}_k = \log_2\left(\frac{\theta_{\text{DTA},k}}{\theta_{\text{CTL},k}}\right)$$

**解釈**：
- Log2(FC) > 0：DTA群で細胞タイプ$k$が増加
- Log2(FC) < 0：DTA群で細胞タイプ$k$が減少
- Log2(FC) = 0：群間で差なし

**比較の特徴**：
- **事後分布ベース**：点推定ではなく、不確実性を考慮した分布として比較
- **ベイズ信用区間**：89%信用区間により統計的有意性を判定

---

### 4. 統計処理方法（Statistical Analysis Method）

#### 4.1 なぜ従来の統計手法では不十分か？

| 統計手法 | 問題点 |
|---------|--------|
| **t検定** | 構成データの制約を考慮できない。見かけ上の負の相関が生じる。 |
| **カイ二乗検定** | 細胞タイプ間の依存関係を考慮できない。多重比較の問題。 |
| **Mann-Whitney U検定** | サンプル間のばらつきを適切にモデル化できない。 |

#### 4.2 ディリクレ多項式モデリング（DMM）の利点

| 特徴 | 説明 |
|------|------|
| **構成データの制約を考慮** | 負の共分散構造を統計的に正しく扱う |
| **階層的モデリング** | サンプル内の変動と群内の変動を同時にモデル化 |
| **ベイズ推定** | 不確実性を定量化し、信用区間を計算可能 |
| **柔軟性** | サンプル数や細胞タイプ数の変動に対応可能 |

#### 4.3 統計処理のステップ

```
ステップ1: データ準備
  ↓
  各サンプルの細胞タイプ別細胞数をカウント
  ↓
  サンプル×細胞タイプのカウントマトリックスを作成

ステップ2: モデル定義
  ↓
  多項分布：サンプル内の変動をモデル化
  ↓
  ディリクレ分布：群内の変動をモデル化

ステップ3: パラメータ推定（HMC法）
  ↓
  各群について独立に事後分布を推定
  ↓
  各細胞タイプの比率θの事後分布を取得

ステップ4: 群間比較
  ↓
  Log2(FC)の事後分布を計算
  ↓
  89%ベイズ信用区間を計算

ステップ5: 有意性判定
  ↓
  信用区間が0を含まない場合に有意と判定
```

#### 4.4 統計処理のパラメータ設定

| パラメータ | 設定値 | 説明 |
|-----------|--------|------|
| **事前分布（α）** | 10⁻⁷ | 弱い情報事前分布（均一な事前分布） |
| **チェーン数** | 4 | MCMCチェーンの数 |
| **反復回数** | 4,000回 | 各チェーンのサンプリング回数 |
| **ウォームアップ** | 2,000回 | バーンイン期間 |
| **適応パラメータ** | adapt_delta = 0.95 | HMCのステップサイズ調整 |
| **信用区間** | 89% | ベイズ信用区間の信頼度 |
