# クラスターアノテーション

## 内容

- **FindAllMarkers_クラスターマーカー同定スクリプト.R** - クラスターマーカー遺伝子同定スクリプト
- **FindAllMarkers_CTL_結果.csv** - CTL条件のFindAllMarkers結果
- **FindAllMarkers_DTA_結果.csv** - DTA条件のFindAllMarkers結果
- **クラスター細胞型対応_CTL.csv** - CTLのクラスターと細胞型の対応表
- **クラスター細胞型対応_DTA.csv** - DTAのクラスターと細胞型の対応表
- **各クラスター上位4マーカー_CTL.csv** - CTLの各クラスター上位4マーカー
- **各クラスター上位4マーカー_DTA.csv** - DTAの各クラスター上位4マーカー
- **FindAllMarkers結果の読み方ガイド.md** - 結果の読み方ガイド

## 説明

FindAllMarkersを用いて各クラスターのマーカー遺伝子を同定し、クラスターに細胞型アノテーションを付与した結果です。
