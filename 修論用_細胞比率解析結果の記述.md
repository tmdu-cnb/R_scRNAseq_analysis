# 修論用：細胞タイプ比率解析結果の記述

## 【修論本文用】細胞タイプ比率解析結果セクション

次に、DTAモデルにおける小脳細胞組成の変化を明らかにするため、DTA群と対照群（CTL）間で各細胞タイプの比率を比較した。細胞タイプ比率は構成データの性質を持ち、各要素の合計が一定（100%）という制約により、ある細胞タイプの比率が増加すると他の細胞タイプの比率は必ず減少するという負の共分散構造が生じる。この制約により、従来の統計手法（t検定、カイ二乗検定など）では、細胞タイプ間の依存関係を適切に考慮できない。そこで本研究では、各サンプル（n=3 per group）での細胞タイプ構成比を計算し、効果量（effect size）のCohen's dを用いて群間差を評価した。Cohen's dは、サンプルサイズに依存せず実際の差の大きさを評価できる指標であり、小サンプルサイズ（n=3）においても各細胞タイプの効果量を評価することができる。

各サンプルにおいて、10種類の小脳細胞タイプ（顆粒細胞、プルキンエ細胞、ベルクマングリア、介在ニューロン、アストロサイト、オリゴデンドロサイト、OPC、ミクログリア、内皮細胞、線維芽細胞）の細胞数をカウントし、各サンプルでの構成比を計算した。DTA群とCTL群の各細胞タイプの構成比の平均値と標準偏差から、Cohen's dを計算した。Cohen's dの解釈は、Cohen (1988)の基準に従い、|d| < 0.2をnegligible、0.2 ≤ |d| < 0.5をsmall、0.5 ≤ |d| < 0.8をmedium、|d| ≥ 0.8をlarge effectとした。

### 細胞タイプマーカー遺伝子の発現パターン

対照群（CTL）における各細胞タイプのマーカー遺伝子の発現パターンを可視化するため、各細胞タイプごとに4つの主要マーカー遺伝子を選択し、ヒートマップを作成した（図○）。使用したマーカー遺伝子は以下の通りである：顆粒細胞（ETV1, RIMS1, SCN2A, GRM4）、オリゴデンドロサイト（PLP1, MBP, MOBP, CNP）、ベルクマングリア（SLC1A3, NDRG2, AQP4, TUBB2B）、アストロサイト（TTN, GFAP, ALDH1L1, S100β (S100B)）、介在ニューロン（KIT, GAD1, GAD2, PVALB）、内皮細胞（CLDN5, CD31 (PECAM1), VWF, FLT1）、プルキンエ細胞（ITPR1, CALB1, PCP4, CA8 (CAR8)）、OPC（PTPRZ1, VCAN, TNR, PDGFRA）、ミクログリア（CD74, C1QB, C3, CSF1R）、線維芽細胞（DCN, APOD, TAGLN, FBLN1）。

ヒートマップでは、各遺伝子の発現量を対数正規化された発現値からz-scoreスケーリング（中央揃えおよびスケール表示）により標準化し、青（低発現）から白（平均発現、z-score = 0）を経て赤（高発現）のカラースケールで表示した。各細胞タイプのマーカー遺伝子は、その細胞タイプにおいて最も高い発現を示し、対角線状のパターンが観察された。これは、選択したマーカー遺伝子が各細胞タイプに特異的に発現していることを示している。

### 全サンプルを統合した細胞タイプ構成比

DTA群とCTL群の全サンプルを統合した細胞タイプ構成比を比較した（図○）。DTA群では、顆粒細胞が62.08%（16,792 cells / 27,050 cells）を占め、最も多い細胞タイプであった。次いで、ベルクマングリアが21.11%（5,710 cells）、オリゴデンドロサイトが6.26%（1,693 cells）、介在ニューロンが4.43%（1,199 cells）、プルキンエ細胞が2.89%（782 cells）の順であった。一方、CTL群では、顆粒細胞が74.87%（19,683 cells / 26,288 cells）を占め、次いでオリゴデンドロサイトが8.54%（2,244 cells）、ベルクマングリアが7.38%（1,941 cells）、介在ニューロンが4.51%（1,186 cells）、プルキンエ細胞が1.27%（335 cells）の順であった。この比較から、DTA群では顆粒細胞の構成比が減少し（62.08% vs 74.87%）、一方でベルクマングリアの構成比が増加している（21.11% vs 7.38%）ことが視覚的に確認できる。

### サンプル単位での効果量評価

各サンプル（n=3 per group）での細胞タイプ構成比の平均値と標準偏差からCohen's dを計算し、効果量を評価した。その結果、**顆粒細胞**はDTA群において大きな効果量で減少していた（Cohen's d = -1.32, large effect）。一方で、**プルキンエ細胞**については大きな効果量で増加していた（Cohen's d = +1.03, large effect）。これは、個々のサンプルのばらつきや希少細胞タイプの変化を検出する力の低下によるものと考えられる。さらに、**アストロサイト**（Cohen's d = +0.96, large effect）、**ベルクマングリア**（Cohen's d = +1.33, large effect）、**OPC**（Cohen's d = +1.08, large effect）、**線維芽細胞**（Cohen's d = +1.28, large effect）は、対照群と比較してDTA群で大きな効果量で増加を示した。一方で、**オリゴデンドロサイト**（Cohen's d = -1.14, large effect）と**内皮細胞**（Cohen's d = -2.35, large effect）は、対照群と比較してDTA群で大きな効果量で減少していた。**介在ニューロン**（Cohen's d = -0.25, small effect）と**ミクログリア**（Cohen's d = -0.31, small effect）については、小さな効果量で減少傾向が認められた。

これらの結果を総合すると、DTAモデルにおける小脳の細胞組成の変化が明らかとなった。顆粒細胞の減少とベルクマングリア、アストロサイトの増加は、小脳における神経細胞の損失とグリア細胞の反応性変化を示唆している。また、オリゴデンドロサイトと内皮細胞の減少は、白質の変性や血管系の変化を示している可能性がある。なお、構成データの制約により、各細胞タイプの変化は独立ではなく、ある細胞タイプの増加は他の細胞タイプの減少と関連していることに注意が必要である。

## 【補足】論文との比較

### 論文の結果（DMM-HMC、89%信頼区間）

- **顆粒細胞**：有意に減少
- **プルキンエ細胞**：有意ではないが減少傾向
- **アストロサイト、ベルクマングリア、ミクログリア、オリゴデンドロサイト**：有意に増加

### 本研究の結果（Cohen's d、n=3 vs n=3）

- **顆粒細胞**：Cohen's d = -1.32（large effect、減少）
- **プルキンエ細胞**：Cohen's d = +1.03（large effect、増加）※論文とは逆
- **アストロサイト**：Cohen's d = +0.96（large effect、増加）
- **ベルクマングリア**：Cohen's d = +1.33（large effect、増加）
- **オリゴデンドロサイト**：Cohen's d = -1.14（large effect、減少）※論文とは逆
- **ミクログリア**：Cohen's d = -0.31（small effect、減少傾向）※論文とは逆
- **内皮細胞**：Cohen's d = -2.35（large effect、減少）※論文には記載なし

### 違いの理由

1. **統計手法の違い**：論文はDMM-HMC（構成データの制約を考慮）、本研究はCohen's d（各細胞タイプを独立に評価）
2. **サンプルサイズ**：論文はより大きなサンプルサイズ、本研究はn=3
3. **検出力**：サンプルサイズが小さいため、統計的有意性は検出されていないが、効果量は大きい

## 【番外編】細胞タイプ分類アルゴリズム

### 分類方法

本研究では、10種類の小脳細胞タイプの分類に、**マーカー遺伝子ベースのスコアリング方式**を採用した。この方法は、各細胞タイプに特異的に発現するマーカー遺伝子の組み合わせを用いて、各細胞の遺伝子発現パターンから細胞タイプを決定する。

### 使用したマーカー遺伝子

各細胞タイプに対し、論文に基づいて以下のマーカー遺伝子を定義した：

1. **顆粒細胞（Granule cells）**: RIMS1, GRM4, GABRA6
2. **プルキンエ細胞（Purkinje cells）**: ITPR1, CALB1, CAR8
3. **ベルクマングリア（Bergmann glia）**: TUBB2B, AQP4
4. **介在ニューロン（Interneurons）**: GAD1, PVALB
5. **アストロサイト（Astrocytes）**: TTN, AQP1, GFAP
6. **オリゴデンドロサイト（Oligodendrocytes）**: PLP1, MBP
7. **OPC（Oligodendrocyte precursor cells）**: PDGFRA, OLIG1
8. **ミクログリア（Microglia）**: CD74, CSF1R
9. **内皮細胞（Endothelial cells）**: CLDN5, VWF
10. **線維芽細胞（Fibroblasts）**: DCN, APOD

### アルゴリズムの流れ

#### ステップ1：各細胞タイプのスコア計算

各細胞について、各細胞タイプのマーカー遺伝子の正規化発現量（normalized expression）の平均値を計算し、それをその細胞タイプのスコアとする。

$$\text{スコア}_{\text{細胞タイプ}i} = \frac{1}{n}\sum_{j=1}^{n} \text{正規化発現量}(\text{マーカー遺伝子}_j)$$

ここで、$n$はその細胞タイプのマーカー遺伝子数である。

**具体例**：
- 顆粒細胞のスコア = mean(RIMS1の発現量, GRM4の発現量, GABRA6の発現量)
- プルキンエ細胞のスコア = mean(ITPR1の発現量, CALB1の発現量, CAR8の発現量)

#### ステップ2：最大スコアによる分類

各細胞について、10種類の細胞タイプのスコアを計算した後、**最大スコアを持つ細胞タイプ**をその細胞の分類結果とする。

$$\text{細胞タイプ} = \arg\max_{i} \text{スコア}_{\text{細胞タイプ}i}$$

この方法により、すべての細胞が必ず10種類のいずれかに分類される（閾値なし）。

### アルゴリズムの特徴

#### 利点

1. **シンプルで理解しやすい**：マーカー遺伝子の発現量の平均を計算するだけ
2. **論文準拠**：既存の論文で報告されているマーカー遺伝子を使用
3. **必ず分類される**：すべての細胞が必ずいずれかの細胞タイプに分類される
4. **計算が高速**：単純な平均値計算のため、処理が高速

#### 限界

1. **排他的分類**：1つの細胞は必ず1つの細胞タイプのみに分類される（混合状態は考慮しない）
2. **マーカー遺伝子の依存性**：分類結果は使用するマーカー遺伝子の選択に大きく依存する
3. **マーカー遺伝子の欠損**：データセットにマーカー遺伝子が含まれていない場合、その細胞タイプの分類精度が低下する可能性がある
4. **閾値なし**：スコアが低くても、最大スコアであれば分類される

### 実装の詳細

本研究では、Seuratオブジェクトの`data`レイヤー（正規化済み発現量）を使用してスコアを計算した。正規化は、Seuratの標準的な正規化手法（LogNormalize）により行われている。

```r
# 正規化データを取得
data_norm <- GetAssayData(seu_obj, layer = "data")

# 各細胞タイプのスコアを計算
scores[, "Granule"] <- colMeans(data_norm[granule_markers, , drop = FALSE])
scores[, "Purkinje"] <- colMeans(data_norm[purkinje_markers, , drop = FALSE])
# ... 以下同様

# 最大スコアの細胞タイプを割り当て
max_types <- apply(scores, 1, which.max)
cell_type <- colnames(scores)[max_types]
```

### 代替手法との比較

本研究で採用したマーカー遺伝子ベースの分類は、以下のような代替手法と比較される：

- **SingleR**：参照データセットとの相関に基づく分類（より客観的だが、参照データに依存）
- **クラスタリング + マーカー遺伝子の確認**：まずクラスタリングを行い、その後各クラスターのマーカー遺伝子を確認（より探索的）
- **機械学習ベースの分類**：分類器を訓練して分類（より複雑だが、高精度）

本研究では、既存の論文との比較可能性を重視し、マーカー遺伝子ベースの分類を採用した。
