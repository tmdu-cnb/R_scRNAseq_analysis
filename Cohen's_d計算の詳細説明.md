# Cohen's d計算の詳細：構成比に対する計算

## 質問：Cohen's dの計算は構成比に対して行われているか？

### 答え：**はい、構成比（Proportion）に対して計算されています**

---

## ① 実際の計算過程

### ステップ1：各サンプルでの構成比を取得

コードの該当部分：
```r
# DTA群の割合（構成比）
dta_props <- prop_by_sample_df$Proportion[
  prop_by_sample_df$Group == "DTA" & prop_by_sample_df$CellType == celltype
]

# CTL群の割合（構成比）
ctl_props <- prop_by_sample_df$Proportion[
  prop_by_sample_df$Group == "CTL" & prop_by_sample_df$CellType == celltype
]
```

**`Proportion`は構成比（0-1の値）です**

### ステップ2：平均と標準偏差を計算

```r
DTA_mean = mean(dta_props)  # 構成比の平均値
DTA_sd = sd(dta_props)      # 構成比の標準偏差
CTL_mean = mean(ctl_props)  # 構成比の平均値
CTL_sd = sd(ctl_props)      # 構成比の標準偏差
```

### ステップ3：Cohen's dを計算

```r
Cohens_d <- (DTA_mean - CTL_mean) / pooled_sd
```

ここで、`DTA_mean`と`CTL_mean`は**構成比の平均値**です。

## ② 具体例：顆粒細胞の場合

### 各サンプルでの構成比（Proportion）

**DTA群の3サンプル**：
- サンプル1: 0.5943（59.43%）
- サンプル2: 0.4984（49.84%）
- サンプル3: 0.7639（76.39%）

**CTL群の3サンプル**：
- サンプル1: 0.7614（76.14%）
- サンプル2: 0.7152（71.52%）
- サンプル3: 0.7658（76.58%）

### 平均と標準偏差（構成比の値）

- **DTA_mean** = mean(0.5943, 0.4984, 0.7639) = **0.6188**（構成比）
- **DTA_sd** = sd(0.5943, 0.4984, 0.7639) = **0.1344**（構成比の標準偏差）
- **CTL_mean** = mean(0.7614, 0.7152, 0.7658) = **0.7474**（構成比）
- **CTL_sd** = sd(0.7614, 0.7152, 0.7658) = **0.0280**（構成比の標準偏差）

### Cohen's dの計算

$$d = \frac{0.6188 - 0.7474}{s_{\text{pooled}}} = \frac{-0.1286}{s_{\text{pooled}}}$$

ここで、$s_{\text{pooled}}$も構成比の標準偏差から計算されます。

## ④ まとめ

### ✅ Cohen's dは構成比に対して計算されている

1. **入力データ**：各サンプルでの構成比（Proportion: 0-1の値）
2. **平均値**：構成比の平均値（例：0.6188 = 61.88%）
3. **標準偏差**：構成比の標準偏差
4. **Cohen's d**：構成比の差を標準化した効果量

### 重要なポイント

- **構成比（Proportion）**：0-1の値、合計が1になる制約がある
- **絶対数（Count）**：細胞数、制約なし

本研究では、**構成比（Proportion）**に対してCohen's dを計算しています。

---

## ⑤ 構成比で計算することの意味

### 構成比で計算する利点

1. **サンプル間の比較が可能**：各サンプルの総細胞数が異なっても比較できる
2. **標準化された値**：0-1の範囲で統一されている
3. **生物学的な意味**：「全体に占める割合」という解釈ができる

### 構成比で計算することの注意点

1. **構成データの制約**：合計が1になるため、細胞タイプ間は相関している
2. **標準偏差の解釈**：構成比の標準偏差は、絶対数の標準偏差とは異なる意味を持つ

---

## 結論

**Cohen's dは構成比（Proportion）に対して計算されています。**

- 入力：各サンプルでの構成比（0-1の値）
- 計算：構成比の平均と標準偏差を使用
- 出力：構成比の差を標準化した効果量
