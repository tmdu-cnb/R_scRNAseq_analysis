# 修論用：Figure 2の説明（細胞タイプ解析）

## Figure 2. DTAモデルと対照群における小脳細胞タイプの解析（snRNA-seq）

### (A) 主要細胞タイプのUMAP次元削減プロット

DTAモデルと対照群（CTL）の小脳において、snRNA-seqにより同定された主要細胞タイプのUMAP（Uniform Manifold Approximation and Projection）次元削減プロットを示した。各条件について、10種類の小脳細胞タイプ（顆粒細胞、プルキンエ細胞、ベルクマングリア、介在ニューロン、アストロサイト、オリゴデンドロサイト、OPC、ミクログリア、内皮細胞、線維芽細胞）が色分けされて表示されている。UMAPは、高次元の遺伝子発現空間を2次元に投影し、類似した遺伝子発現パターンを持つ細胞が近くに配置されるように可視化した。

### (B) 対照群小脳における各主要細胞タイプのマーカー遺伝子

対照群（CTL）小脳における各主要細胞タイプごとの4つの主要マーカー遺伝子をヒートマップで表示した。ヒートマップは、対数正規化された発現値を中央揃えおよびスケール表示（z-scoreスケーリング）した値で示している。使用したマーカー遺伝子は以下の通りである：顆粒細胞（ETV1, RIMS1, SCN2A, GRM4）、オリゴデンドロサイト（PLP1, MBP, MOBP, CNP）、ベルクマングリア（SLC1A3, NDRG2, AQP4, TUBB2B）、アストロサイト（TTN, GFAP, ALDH1L1, S100β (S100B)）、介在ニューロン（KIT, GAD1, GAD2, PVALB）、内皮細胞（CLDN5, CD31 (PECAM1), VWF, FLT1）、プルキンエ細胞（ITPR1, CALB1, PCP4, CA8 (CAR8)）、OPC（PTPRZ1, VCAN, TNR, PDGFRA）、ミクログリア（CD74, C1QB, C3, CSF1R）、線維芽細胞（DCN, APOD, TAGLN, FBLN1）。各細胞タイプのマーカー遺伝子は、その細胞タイプにおいて最も高い発現を示し、対角線状のパターンが観察された。

### (C) DTAモデルと対照群小脳における細胞タイプ比率

DTAモデルと対照群（CTL）小脳における細胞タイプの比率を積み上げ棒グラフで比較した。DTA群では顆粒細胞が62.08%（16,792 cells / 27,050 cells）を占め、CTL群では74.87%（19,683 cells / 26,288 cells）を占めた。一方、ベルクマングリアはDTA群で21.11%（5,710 cells）、CTL群で7.38%（1,941 cells）と、DTA群で大きく増加していた。オリゴデンドロサイトはDTA群で6.26%（1,693 cells）、CTL群で8.54%（2,244 cells）と減少していた。

### (D) DTAモデルと対照群における細胞タイプの相対的存在量

DTAモデルと対照群（CTL）における各細胞タイプの相対的存在量を、効果量（Cohen's d）により評価した。各サンプル（n=3 per group）での細胞タイプ構成比からCohen's dを計算し、効果量の大きさを評価した。その結果、顆粒細胞はDTA群において大きな効果量で減少していた（Cohen's d = -1.32, large effect）。一方で、プルキンエ細胞は大きな効果量で増加していた（Cohen's d = +1.03, large effect）。さらに、アストロサイト（Cohen's d = +0.96, large effect）、ベルクマングリア（Cohen's d = +1.33, large effect）、OPC（Cohen's d = +1.08, large effect）、線維芽細胞（Cohen's d = +1.28, large effect）は、対照群と比較してDTA群で大きな効果量で増加を示した。一方で、オリゴデンドロサイト（Cohen's d = -1.14, large effect）と内皮細胞（Cohen's d = -2.35, large effect）は、対照群と比較してDTA群で大きな効果量で減少していた。効果量の解釈は、Cohen (1988)の基準に従い、|d| ≥ 0.8をlarge effectとした。

## 【簡潔版】Figure 2の説明

### Figure 2. DTAモデルと対照群における小脳細胞タイプの解析（snRNA-seq）

**(A)** DTAモデルと対照群（CTL）の小脳において、snRNA-seqにより同定された主要細胞タイプのUMAP次元削減プロット。10種類の小脳細胞タイプが色分けされて表示されている。

**(B)** 対照群（CTL）小脳における各主要細胞タイプごとの4つの主要マーカー遺伝子。ヒートマップは、対数正規化された発現値を中央揃えおよびスケール表示（z-scoreスケーリング）した値で示している。

**(C)** DTAモデルと対照群（CTL）小脳における細胞タイプの比率。積み上げ棒グラフにより、各細胞タイプの構成比を比較した。

**(D)** DTAモデルと対照群（CTL）における各細胞タイプの相対的存在量。効果量（Cohen's d）により評価し、|d| ≥ 0.8をlarge effectとした。顆粒細胞はDTA群で大きな効果量で減少し（Cohen's d = -1.32）、ベルクマングリアは大きな効果量で増加していた（Cohen's d = +1.33）。

## 【補足】論文との違い

### 論文（A-T vs Control）
- **(A)**: 各条件10,000細胞にダウンサンプリング
- **(B)**: 各細胞タイプの上位5つのマーカー遺伝子
- **(C)**: 積み上げ棒グラフで構成比を表示
- **(D)**: DMM-HMCによる事後分布、89%信頼区間、log2(proportion in A-T/proportion in control)

### 本研究（DTA vs CTL）
- **(A)**: 全細胞を使用（ダウンサンプリングなし）
- **(B)**: 各細胞タイプの4つの主要マーカー遺伝子
- **(C)**: 積み上げ棒グラフで構成比を表示（論文と同様）
- **(D)**: Cohen's dによる効果量評価（DMM-HMCではなく、サンプルサイズが小さいため）
