# Cohen's d効果量の原理説明

## 効果量（Effect Size）とは？

効果量は、統計的有意性（p値）とは別に、**実際の差の大きさ**を評価する指標です。p値はサンプルサイズに依存しますが、効果量はサンプルサイズに依存せず、**実際の生物学的な意味のある差**を評価できます。

## Cohen's dとは？

**Cohen's d**は、2群間の平均値の差を標準偏差で標準化した効果量です。

### 計算式

$$d = \frac{\bar{X}_1 - \bar{X}_2}{s_{\text{pooled}}}$$

ここで：
- $\bar{X}_1$ = 群1（DTA群）の**各サンプルでの構成比の平均値**
- $\bar{X}_2$ = 群2（CTL群）の**各サンプルでの構成比の平均値**
- $s_{\text{pooled}}$ = プールされた標準偏差（pooled standard deviation）

### 「平均値」とは何か？

本研究では、**各サンプル（n=3）での細胞タイプ構成比の平均値**を指します。

**具体例：顆粒細胞の場合**

DTA群の3サンプルでの構成比：
- サンプル1（7928-DTA_dta_1）: 59.43%
- サンプル2（8112-DD3）: 49.84%
- サンプル3（8696-DD2）: 76.39%

**DTA群の平均値** = (59.43% + 49.84% + 76.39%) / 3 = **61.88%**

CTL群の3サンプルでの構成比：
- サンプル1（8693-DC1）: 76.14%
- サンプル2（8694-DC2）: 71.52%
- サンプル3（8695-DC3）: 76.58%

**CTL群の平均値** = (76.14% + 71.52% + 76.58%) / 3 = **74.74%**

つまり、**$\bar{X}_1$ = 0.6188（DTA群の平均構成比）**、**$\bar{X}_2$ = 0.7474（CTL群の平均構成比）**です。

### プールされた標準偏差の計算

$$s_{\text{pooled}} = \sqrt{\frac{(n_1 - 1)s_1^2 + (n_2 - 1)s_2^2}{n_1 + n_2 - 2}}$$

ここで：
- $n_1$ = 群1のサンプル数
- $n_2$ = 群2のサンプル数
- $s_1$ = 群1の標準偏差
- $s_2$ = 群2の標準偏差

## Cohen's dの解釈

Cohen (1988)による基準：

| Cohen's d | 効果量の大きさ | 解釈 |
|-----------|---------------|------|
| |d| < 0.2 | 小さい（negligible） | 実質的に差がない |
| 0.2 ≤ |d| < 0.5 | 小さい（small） | 小さな差 |
| 0.5 ≤ |d| < 0.8 | 中程度（medium） | 中程度の差 |
| |d| ≥ 0.8 | 大きい（large） | 大きな差 |

### 符号の意味

- **d > 0**: DTA群の方が大きい（増加）
- **d < 0**: CTL群の方が大きい（減少）

## なぜCohen's dを使うのか？

### ① サンプルサイズに依存しない

p値はサンプルサイズが大きくなると小さくなりやすいですが、Cohen's dはサンプルサイズに依存しません。

### ② 実際の差の大きさを評価

例：
- **d = 0.3**: 平均値の差が標準偏差の0.3倍 → 小さな差
- **d = 1.0**: 平均値の差が標準偏差の1.0倍 → 大きな差

### ③ 生物学的な意味を理解しやすい

「統計的に有意（p < 0.05）」でも、実際の差が小さい（d < 0.2）場合は、生物学的には意味がない可能性があります。

## 本研究での適用

各細胞タイプの構成比について：

1. **各サンプルでの構成比を計算**（n=3 per group）
2. **DTA群とCTL群の平均と標準偏差を計算**
3. **Cohen's dを計算**
4. **効果量の大きさを評価**

### 具体例

顆粒細胞の場合：
- DTA群: 平均 = 0.619, SD = 0.134
- CTL群: 平均 = 0.747, SD = 0.028

$$s_{\text{pooled}} = \sqrt{\frac{(3-1) \times 0.134^2 + (3-1) \times 0.028^2}{3 + 3 - 2}} = \sqrt{\frac{2 \times 0.0180 + 2 \times 0.0008}{4}} = 0.097$$

$$d = \frac{0.619 - 0.747}{0.097} = -1.32$$

**解釈**: |d| = 1.32 > 0.8 → **大きな効果量**。DTA群で顆粒細胞が大きく減少している。

## Cohen's dの利点と限界

### 利点

1. **直感的な解釈**: 標準偏差の何倍の差かを示す
2. **サンプルサイズに依存しない**: 小サンプルでも評価可能
3. **メタ解析に適している**: 異なる研究間で比較可能

### 限界

1. **サンプルサイズが小さい場合**: 標準偏差の推定が不安定になる可能性
2. **構成データへの適用**: 構成データ（合計が1）の場合、標準偏差の解釈に注意が必要

## 構成データへの適用時の注意点

構成データ（細胞タイプ比率）の場合：
- 各細胞タイプの比率は**相関している**（合計が1）
- Cohen's dは各細胞タイプを**独立に**評価する
- 全体の構成の変化を評価するには、複数の細胞タイプを同時に考慮する方法（DMMなど）がより適切

しかし、**各細胞タイプごとの効果量を評価する**という目的では、Cohen's dは有用です。
